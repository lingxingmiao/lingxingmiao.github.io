SEUS PTGI HRR Test 2.1 GFME 基于 SEUS PTGI 的大型魔改光影

使用方式：

1.需要拥有一个原版（未经任何修改）的 SEUS PTGI HRR Test 2.1.zip 文件。文件名必须完全匹配，若下载时空格被其他符号替换，请手动重命名文件。

2.将 SEUS PTGI HRR Test 2.1.zip 和本 jar 置于同一目录。双击运行 jar 后，若一切正常，应该会在同目录生成本光影的文件。后续安装与使用与一般光影无异。

3.若不能正常输出光影文件，请使用命令行程序手动执行 java -jar "jar路径"。jar 程序会输出碰到的异常。

参数：

  • 阴影精度：4k-32k
  • 阴影范围：80-640
  • 光追范围：60-320
  • 光追采样：1-5

最低显存要求：3GB
